## 使用方式

**自动路由**：直接描述你的问题，系统自动匹配最合适的顾问。

**指定顾问**：说「让芒格来」「用费曼方式」直接激活指定顾问。

**圆桌模式**：说「开个圆桌」「让他们辩一辩」启动多顾问讨论。
